 
# 🐍 Python Snake Game

A classic Snake game built with Python using the Turtle graphics library. Features high score persistence across sessions and smooth snake movement.

## 🎮 Gameplay

- Control the snake using **Arrow Keys**
- Eat the blue food to grow longer and earn points
- Avoid hitting the **walls** or your own **tail**
- Your high score is saved automatically between sessions!

## 📸 Features

- ✅ Smooth snake movement with segment-following logic
- ✅ High score saved to `data.txt` — persists after closing
- ✅ Snake resets on collision (wall or tail) without closing the game
- ✅ Score and High Score displayed on screen
- ✅ Lime-colored snake head for easy visibility

## 🚀 How to Run

1. Make sure **Python** is installed on your system
2. Clone this repository:
